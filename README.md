# Global Electronics Retailer: E-Commerce Sales & Customer Feedback Analysis

## 📈 Business Overview
This project focuses on analyzing sales performance and customer feedback for a global electronics retailer. The goal was to transform raw transactional and feedback data into actionable strategic insights to optimize supply chain efficiency, improve customer retention, and drive revenue growth.

## 🧠 Key Business Insights & Impact
* **Sales Optimization:** Identified key product categories driving 80% of revenue, allowing for better inventory allocation.
* **Customer Feedback Loop:** Analyzed customer sentiment trends to pinpoint friction points in delivery and product quality, leading to actionable recommendations for operational improvements.
* **Regional Demand:** Mapped out high-performing regions to help target marketing spend more effectively.

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Plotly / Matplotlib / Seaborn

## 📂 Project Structure
* `notebooks/`: Contains the full exploratory data analysis (EDA) and visualization pipeline.
* `data/`: Sample dataset (excluding sensitive production data).
